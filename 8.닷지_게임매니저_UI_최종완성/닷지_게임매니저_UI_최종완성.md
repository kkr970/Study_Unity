### 게임 UI 제작
유니티의 UI는 게임월드와 별개의 공간이 아닌 하나의 오브젝트로 구성되어짐  
게임 오브젝트처럼 UI오브젝트를 다룰 수 있음  
* EventSystem 게임 오브젝트
  이벤트 시스템 컴포넌트를 가진 게임 오브젝트  
  UI 게임 오브젝트에 터치, 그래드 등 상호작용을 이벤트 메시지로 전달함  
* UI 배치
  UI 배치에서는 앵커, 피벗, 포지션을 지정해줘야함
